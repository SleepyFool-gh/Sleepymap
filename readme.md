# Sleepy Macros — `Areamap` library

A map library for SugarCube designed for room-to-room movement like **Darkest Dungeon** or node-to-node movement like **Faster Than Light** — *NOT* for grid movement like **Zelda** or **Final Fantasy Tactics**. `Areamap` takes a space-separated 2D text grid and converts it into a functional map for player navigation. Included are various built-in macros and methods to interact and display the map.

[Read the documentation here](https://sleepyfool-gh.github.io/areamap/)

- [areamap.js](./dist/areamap.js) — Main library definitions
- [areamap.css](./dist/areamap.css) — Library styling
- [ArgObj.js](./dist/ArgObj.js) — Required argument parsing dependency
